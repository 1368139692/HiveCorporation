# 安装&使用(Install&Use)
### 前置插件(Reliance)
1. Slimefun 粘液科技
2. RykenSlimefunCustomizer 粘液自定义插件
3. Skript 脚本插件
### 本体下载(Download)
1. GitHub
2. 群聊文件分享：895650188
### 安装方式(Install)
1. 安装前置插件，并启动服务端。
2. 下载插件数据，并将其解压后置于'plugins/RykenSlimeCustomier/addons'中。
3. 将文件中HiveCorporation.sk文件置于'plugins/Skript/scripts'中
4. 重启服务端
